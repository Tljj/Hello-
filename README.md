---

## 🎯 课程亮点

*   **🛠️ 工具驱动**：基于业界流行的 **MeterSphere** 开源测试平台与 **Halo** 博客系统，所学即所用。
*   **🤖 AI赋能**：深度融合 AI 辅助测试与学习，教你如何与AI协作，提升测试效率与深度。
*   **📚 项目实战**：理论结合实践，通过一个完整的“Halo博客系统测试项目”，贯穿测试全流程。
*   **🌐 开源开放**：课程本身即开源项目，你将亲身体验开源协作，你的贡献将永久留存在项目历史中。

---

## 📖 课程目录

| 阶段 | 核心内容 | 课时 |
| :--- | :--- | :--- |
| **一、奠基与启动** | 环境搭建、测试基础概念、初识MeterSphere与Halo | 4课时 |
| **二、核心技能实战** | 功能测试、测试用例设计、缺陷管理、接口测试、UI自动化 | 16课时 |
| **三、AI赋能与提升** | AI辅助测试设计、测试分析与调试、报告撰写 | 8课时 |
| **四、综合复盘与展望** | 性能测试初探、毕业项目答辩与总结 | 4课时 |
